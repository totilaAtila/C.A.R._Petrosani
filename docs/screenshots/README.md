# Capturi de ecran

README-ul principal afișează trei imagini din acest folder. Salvează capturile
tale reale (din aplicația pornită) cu exact aceste nume:

| Fișier | Ecran |
|---|---|
| `statistici.png` | Tabloul de bord **Statistici** (cardurile cu totaluri) |
| `generare-luna.png` | Ecranul **Generare lună** (jurnalul + butoanele) |
| `salvari.png` | Ecranul **Salvări / Managementul bazelor de date** |

Format recomandat: PNG, lățime ~1300 px. După ce le adaugi și faci commit, se
afișează automat în pagina principală (README).
